## 2. OpenClaw 正式日报效果

OpenClaw 的定位是正式日报层。它会基于少量入选条目，输出更适合阅读和分享的正式日报文档。

示例文档：

- [OpenClaw 正式日报示例（飞书 Docx）](https://mnc4pihca8.feishu.cn/docx/A36tdSntHoAFtGxdBetcygdDnQb?from=from_copylink)

特点：

- 更适合做正式整理和沉淀
- 适合发群、留档、复盘
- 比 Hermes 快报更接近“正式内容产物”

## 3. 两层输出的角色分工

### Hermes

- 快
- 轻
- 及时
- 告诉你“这一轮最值得关注什么”

### OpenClaw

- 稍慢
- 更正式
- 更适合沉淀
- 告诉你“今天最终值得保留下来的内容是什么”

## 一句话理解

如果只想快速掌握信息，看 Hermes。  
如果要分享、留档、做正式阅读，看 OpenClaw。
